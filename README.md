## 使用方式

### 离线使用

1. 双击 `index.html`，浏览器打开
2. 选择角色（学生/家长），输入姓名和年级
3. 完成 30 道评估题（约 15 分钟）
4. 查看自动生成的诊断报告（雷达图 + 柱状图 + 处方建议）
5. 数据自动保存在浏览器本地，关闭后不会丢失

### 教师班级管理

1. 点击导航栏「👨‍🏫 教师」，输入密码（默认 `admin123`）
2. 新建班级，自动生成班级码
3. 复制分享链接发送给学生
4. 学生通过链接测评，数据自动归入班级
5. 教师查看班级汇总和导出 CSV

### AI 助手（可选）

1. 注册 [DeepSeek](https://platform.deepseek.com) 获取 API Key
2. 在「🤖 AI助手」页面粘贴 Key
3. 即可与 AI 学伴"小星"对话：学习辅导、情绪支持、报告解读

## 技术说明

- 纯前端单文件，零依赖，无需安装
- 数据存储在浏览器 localStorage，不上传服务器
- 支持 Chrome/Safari/Firefox/Edge
- 手机端自适应
